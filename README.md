## 项目如何使用

``` bash
### 首先下载项目依赖包，使用方命令。
npm install

### 服务器热部署，运行项目
npm run dev

### 项目的构建命令
npm run build

### 以下命令来查看依赖关系.然后再根据具体情况划分代码块.
npm run build --report
